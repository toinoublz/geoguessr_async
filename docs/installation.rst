Installation
============

Requirements
------------

Geoguessr Async requires Python 3.8 or higher.

Install from PyPI
-----------------

.. code-block:: bash

   pip install geoguessr-async

Install from source
-------------------

.. code-block:: bash

   git clone https://github.com/yourusername/geoguessr_async.git
   cd geoguessr_async
   pip install -e .

Dependencies
------------

- `aiohttp` for HTTP requests
- `python-dateutil` for date parsing
